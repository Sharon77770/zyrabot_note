자이라 봇 개인정보 보호정책

본 개인정보 보호정책(이하 "정책")은 자이라 봇(이하 "봇")이 사용자(이하 "사용자")의 개인정보를 어떻게 수집, 사용 및 보호하는지 설명합니다.

1. 수집하는 정보

  • 디스코드 사용자 ID
  • 사용자의 봇 사용 기록
  • 게임 관련 정보 (랭크, 매칭 기록 등)

2. 정보의 사용 목적

  • 팀 매칭 및 밸런스 조정
  • 서비스 개선 및 오류 수정
  • 사용자 문의 대응

3. 정보 공유 및 보호

사용자의 개인정보는 제3자와 공유되지 않으며, 적절한 보안 조치를 통해 보호됩니다.

4. 정책 변경

정책이 변경될 경우 사용자에게 공지됩니다.

Zyra Bot Privacy Policy

This Privacy Policy ("Policy") explains how Zyra Bot ("Bot") collects, uses, and protects user ("User") data.

1. Information Collected

  • Discord User ID
  • Bot usage history
  • Game-related data (rank, match history, etc.)

2. Purpose of Data Usage

  • Team matching and balance adjustments
  • Service improvements and bug fixes
  • Responding to user inquiries

3. Data Sharing and Protection

User data is not shared with third parties and is protected with appropriate security measures.

4. Policy Changes

Users will be notified of any changes to this policy.
